Nome do projeto: Sistema de Música

Descrição do Projeto:

O projeto serve para um Sistema de música.

As suas principais funcionalidades é:

 -Cadastrar uma música;
 -Pesquisar uma música pelo nome e tipo da música;
 -Pesquisar uma música pelo tipo da música;
 -Remover uma música.
